Weather Around The World

Using weather data collected from different cities across the globe, we aim to visualize the relationship between a cities Latitide and the weather it experiences.

Data Collection

Our data is collected from publicly available data through the openWeather API. Visualizations constructed using matplotlib and seaborn python libraries.
